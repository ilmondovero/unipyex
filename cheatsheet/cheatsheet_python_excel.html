3 xlwings — Fondamentali
Aprire Excel
import xlwings as xw

app = xw.App(visible=True)
wb  = app.books.add()          # Nuovo
wb  = xw.Book("file.xlsx")    # Esistente
ws  = wb.sheets[0]            # Primo
ws  = wb.sheets["Nome"]       # Per nome
Leggere e scrivere
# Scrivere
ws.range("A1").value = "Testo"
ws.range("A1").value = 42
ws.range("A1").value = [[1,2],[3,4]] # Matrice

# Scrivere DataFrame
ws.range("A1").options(pd.DataFrame).value = df

# Leggere
val = ws.range("A1").value
df  = ws.range("A1").options(
        pd.DataFrame, expand="table").value
Fogli
ws = wb.sheets.add("NomeFoglio")
ws.name = "NuovoNome"
ws.clear()              # Svuota foglio
ws.delete()             # Elimina
Salvare e chiudere
wb.save("output.xlsx")
wb.close()
app.quit()
Tip: Usa sempre try/finally con app.quit() per evitare processi Excel orfani.
4 xlwings — Formattazione
Font e colori
rng = ws.range("A1")
rng.font.bold  = True
rng.font.size  = 14
rng.font.name  = "Calibri"
rng.font.color = (44, 62, 80)  # RGB

Formato numeri API
# Excel ITALIANO: virgola=decimale, punto=migliaia
rng.api.NumberFormatLocal = "#.##0,00 €"
rng.api.NumberFormatLocal = "0,00%"
rng.api.NumberFormatLocal = "#.##0"
rng.api.NumberFormatLocal = "+0,0%;-0,0%"
rng.api.NumberFormatLocal = "DD/MM/YYYY"
Attenzione! Usa NumberFormatLocal (non number_format) su Excel italiano: "0.0%" con il punto dà 04% invece di 3,5%.
Colonne e merge
ws.autofit()                     # Autofit tutto

5 Formule Italiane
Regola: Usa SEMPRE set_formula(). L'uso diretto di FormulaLocal aggiunge @ davanti alle formule in Excel 365.
Uso corretto
from tw_utils import set_formula

# Corretto
set_formula(ws.range("A1"), "=SOMMA(B1:B10)")

# Sbagliato - aggiunge @ in Excel 365
ws.range("A1").api.FormulaLocal = "=SOMMA(...)"
Separatore: punto e virgola (;)
set_formula(ws.range("A1"),
    '=SE(B1>0;"Positivo";"Negativo")')
set_formula(ws.range("A1"),
    "=CERCA.VERT(A1;D1:E10;2;FALSO)")
set_formula(ws.range("A1"),
    '=CONTA.SE(B1:B10;">0")')
set_formula(ws.range("A1"),
    "=SE.ERRORE(CERCA.X(A1;D:D;E:E);\"\")")

6 Protezione & Export
Celle API
# Blocca / sblocca
ws.range("A1").api.Locked = True
ws.range("B1").api.Locked = False

# Nasconde formula nella barra
ws.range("A1").api.FormulaHidden = True

# Commento su cella
ws.range("A1").api.AddComment("Testo")
Ordine obbligatorio: Prima imposta Locked/FormulaHidden, poi chiama protect_sheet(). Senza protezione attiva, FormulaHidden non ha effetto.
Foglio
from tw_utils import protect_sheet, hide_sheet

# Proteggi con password
protect_sheet(ws, "password")

# Permetti formattazione (es. ES03)
protect_sheet(ws, "password",
              allow_formatting=True)

# Nascondi (very_hidden = non riappare)
hide_sheet(ws, very_hidden=True)
Export PDF
# Tutto il workbook
wb.api.ExportAsFixedFormat(0, "report.pdf")

# Solo un foglio
ws.api.ExportAsFixedFormat(0, "foglio.pdf")
PageSetup API
ps = ws.api.PageSetup
